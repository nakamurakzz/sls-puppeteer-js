# sls-puppeteer-js

## nodejs version
```bash
node -v
v16.19.0
```

## serverless framework インストール
```bash
npm install -g serverless
```

```bash
sls -v
Framework Core: 3.27.0
Plugin: 6.2.3
SDK: 4.3.2
```

## プロジェクトディレクトリに移動
```bash
cd sls-puppeteer-js
```

## モジュールインストール
```bash
npm install
```

## デプロイ
```bash
sls deploy
```
